# 第一个静态博客

&nsop; 时光倒流，第一次使用博客还是在上初中那会，那时候使用 Wordpress 搭建的动态博客，需要使用数据库，需要服务器或者虚拟主机，自己好学，也喜欢建站成功的成就感，博客的内容也不是重点，如今，技术的要求已经不再是我追求的目标，通过使用静态博客，我将着重于博客的内容。
